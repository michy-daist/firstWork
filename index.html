SKILL

HTML

習熟度→★★★★☆


どんなレイアウトでも迷いなくコーディングできるレベル。大半のタグは自由に使いこなせるようになり、保守性の高いコードを書くことに意識を向けられるようになった。

CSS

習熟度→★★★☆☆


LPなどのサイトであれば、ほとんど調べずに模写できるレベル。よく使用されるセレクタやプロパティなどについては理解できているが、深い知識はまだ持っていない。

JavaScript & jQuery

習熟度→★★★☆☆


JavaScriptの基礎文法を用いたjQueryの実装ができるレベル。書いていて一番楽しいと感じる言語。将来的にはNode.jsを学び、サーバーサイドでも使ってみたいと思っている。

PHP

習熟度→★★★☆☆


ページ間のデータ移動や問い合わせフォームの設置、自動メール返信機能を実装できるレベル。選択された対象や個数に応じて金額を計算する簡単なアプリなどであれば作成できる。

MySQL

習熟度→★★☆☆☆


データの追加や削除といった超基礎的なSQL文が書けるレベル。PHPと組み合わせて出席管理のデータベースを作成した経験あり。セキュリティーの知識が浅く、できることは少ない。
